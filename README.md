# Code Lab

Plataforma interativa de ensino de programação e criação de jogos para alunos particulares — desenvolvida por Matheus Marques, Coordenador Pedagógico do CNA.

## Sobre

Material exclusivo voltado para crianças, com trilha progressiva em 4 fases:

- 🎨 **Fase 1 — Arte Digital & Pixel Art** (Piskel)
- 🧩 **Fase 2 — Lógica com Scratch**
- 🎮 **Fase 3 — Jogos 2D com Construct 3**
- 🚀 **Fase 4 — Programação Real com Lua** (Roblox Studio)

## Stack

- Next.js / Vite + React + TypeScript
- Tailwind CSS + shadcn/ui
- SQLite + Drizzle ORM
- Express backend

## Desenvolvimento

```bash
npm install
npm run dev
```

## Acessibilidade

Plataforma projetada com foco em acessibilidade para alunos neurodivergentes — paleta de baixo estímulo, alto contraste, animações reduzidas, navegação clara.
